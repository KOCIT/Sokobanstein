#Sokobanstein 3D

A 3D puzzle-adventure game for Arduboy, blending Sokoban mechanics with raycasting visuals

Описание проекта

Sokobanstein 3D — это уникальная игра для портативной консоли Arduboy, сочетающая классическую механику Sokoban (перемещение ящиков по лабиринту) с псевдотрёхмерной графикой в стиле Wolfenstein 3D. Исследуйте таинственные подвалы Школы 21, решайте пространственные головоломки и помогайте племенам капибар, саламандр, альпак и медоедов найти путь домой.

Особенности

Гибридный геймплей: Переключайтесь между 3D-видом от первого лица и 2D-картой уровня
Три уровня сложности:

Легкий: Полный доступ к 2D-карте с возможностью перемещения
Средний: Только просмотр 2D-карты
Сложный: Игра исключительно в 3D-режиме
4 уникальных уровня: Прогрессивно усложняющиеся головоломки
Система обучения: Пошаговое введение в механику игры
Оригинальный саундтрек: 8-битная музыка и звуковые эффекты
Коллекционные персонажи: Разблокируйте племена животных после каждого уровня
Технические детали

Платформа: Arduboy (ATmega32u4, 128×64 монохромный дисплей)
Библиотеки: Arduboy2, ArduboyTones
Особенности:

Псевдотрёхмерный рендеринг с помощью raycasting
Эффективное использование памяти PROGMEM для графики и уровней
Адаптивная система управления для разных режимов сложности
Минималистичная 8-битная графика в стиле пиксель-арт

Управление

Вперёд/Назад: Крестовина UP/DOWN
Поворот: Крестовина LEFT/RIGHT
Толкнуть ящик: UP + A
Переключить карту: A (в режимах Easy/Medium)
Пауза/Меню: B
Выбор в меню: UP/DOWN + A
История создания

Проект был создан в рамках образовательной программы Школы 21 как демонстрация возможностей Arduboy и креативного подхода к классическим игровым жанрам. Название "Sokobanstein" объединяет Sokoban и Wolfenstein, отражая гибридную природу игры.

Требования для сборки

Arduino IDE 1.8.x или выше

Arduboy2
ArduboyTones
